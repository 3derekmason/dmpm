# dmpm - a Javascript package manager

### Requirements

One of the following versions of [Node.js](https://nodejs.org/en/download/) must be installed to run **`dmpm`**:

- `12.x.x` >= `12.13.0`
- `14.x.x` >= `14.15.0`
- `16.0.0` or higher

### Usage

All `dmpm` packages can be installed with `npm`.

```bash
npm i <command>
```
